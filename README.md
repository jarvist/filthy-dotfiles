Filthy-Dotfiles
===============

Jarv's Filthy Dotfiles, a repository for all those little tweeks between $HOMEs.

==================

git clone https://github.com/jarvist/filthy-dotfiles.git

cd filthy-dotfiles/

. makesymlinks.sh
 
bash
